# DOTS
The Desjeriva Orbital Tracking System makes use of an API from NORAD (North American Aerospace Defense Command) to obtain information about satellites in orbit, and then performs calculations to let you know the trajectory of satellites at your location.

The uses of such a program include, but are not limited to, the calculation of launch windows for rockets and ground based observation for astronomers or hobbyists who use telescopes for visual observation of these satellites for photography.
